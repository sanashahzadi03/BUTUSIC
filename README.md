# BUTUSIC
A Bento-style landing page for an illustration exhibition, featuring a sleek dark theme with high-contrast lime and purple accents. The design uses a modular grid to showcase 3D character art, navigation links, event details, and social proof in a clean, modern interface.

Butusic Exhibition Landing Page A modern, Bento Grid style landing page featuring 3D illustrations and a high-contrast dark theme. This project focuses on clean UI/UX and structured layout design.

✨ Key Features Bento Box Layout: A clean, modular grid structure.

Modern UI: High-contrast colors (Lime, Purple, Black) with a mix of 3D claymorphism.

Responsive Design: Organized sections for navigation, exhibition details, and user ratings.

Call-to-Actions: Integrated QR code and "Best Reviews" buttons.

🛠️ Built With HTML5 - Semantic structure.

CSS3 - Custom Grid and Flexbox layouts.

Google Fonts - Bold typography for an editorial feel.
